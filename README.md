<h1 align="center">
  Roselt Network Tools
</h1>

Cross-Platform Network Utilities App. 

_Note: All tools are still in active development with lots of bugs_

## Current Tools
- Ping IP / Domain

## Planned Tools
- Trace Route
- Port Scanner
- WiFi Scanner
- Network Devices
- Speed Test
- Data Experience Test
- Whois Lookup
- DNS Lookup
- Wake On LAN
- Down Detector
- Total IPs per country/region
- RJ45 Connector Wiring
